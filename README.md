# Social media content analyzer

## Project Description
A MERN stack app that analyzes uploaded social media content and suggests engagement improvements.

## Features
- Login & Signup using JWT
- Upload PDF and images
- OCR and text extraction
- AI-based content analysis
- React SPA frontend
